GoF Designpatterns Examples
===========================

These examples were developed for the "Best Practice Software
Engineering" Book published at Spektrum akademischer Verlag
(Springer), ISBN: 978-3827424860 for the "patterns" chapter. In
this chapter only parts of the sources were printed.

These examples are part of the BPSE Open Source examples. 

The examples include the following patters:

* Interface
* Delegation
* Adapter
* Factory
* Immutable
* Iterator
* Observer
* Proxy
* Singleton

Use Maven 2 to build the examples.

    mvn test

Executes the Unit tests. The Unit tests are part of the examples
and illustrate the usage of certain patterns. Use

    mvn eclipse:eclipse

to generate Eclipse project files.
